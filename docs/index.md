# Gradiverse

解析的に導出できる **勾配・ヤコビアン・ヘッセ（またはHVP）** を、  
**数式（LaTeX）＋実装（コード）＋数値検証（テスト）** で統合するカタログです。

## 目的

- 自動微分なしでも最適化・シミュレーション（FEM含む）を実装できる状態を目指す
- Derivative Card 単位で再利用可能な知識資産を積み上げる

## はじめかた

1. [Derivative Card Standard](/guide/derivative-card-standard) を確認
2. [Testing Standard](/guide/testing-standard) で検証規格を確認
3. [Template](/catalog/templates/derivative-card/spec) を複製して新規カードを作成
